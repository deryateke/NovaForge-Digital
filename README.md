# NovaForge Digital 🔥

NovaForge Digital is a frontend project crafted to look and feel like a real-world agency site. From a slick animated hamburger nav to responsive service cards, a portfolio showcase, client testimonials, and a newsletter form — every section is built with clean SCSS architecture and sharp attention to detail. No frameworks, no shortcuts. Just solid HTML, SCSS, and vanilla JS.
