# NovaForge-Digital



NovaForge Digital is a frontend project crafted to look and feel like a real-world agency site. From a slick animated hamburger nav to responsive service cards, a portfolio showcase, client testimonials, and a newsletter form — every section is built with clean SCSS architecture and sharp attention to detail. No frameworks, no shortcuts. Just solid HTML, SCSS, and vanilla JS.

## 🧩 Sections

| # | Section | What's inside |
|---|---|---|
| 1 | **Header** | Fixed nav, logo, animated hamburger menu |
| 2 | **Hero** | Bold headline, description, CTA buttons |
| 3 | **Services** | UX/UI Design, Growth Marketing, Web Dev, Brand Identity |
| 4 | **About** | Mission, Vision & Values with icons |
| 5 | **Portfolio** | Stats (16 years, 250+ brands), Brand Strategy, Digital Advisory |
| 6 | **Testimonials** | James Carter, Emily Stone, Sophia Martinez feedback cards |
| 7 | **Newsletter** | Email input + subscribe button |
| 8 | **Footer** | Resources, Help, Contact info, Social media links |
